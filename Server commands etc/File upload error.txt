Κατά συνέπεια, το πρόβλημα οφείλεται στο γεγονός οτι το δηλωμένο όριο του χρόνου εκτέλεσης των PHP Scripts ήταν ορισμένος στα 30sec και το συγκεκριμένο script χρειαζόταν περισσότερο για να εκτελεστεί. Για την αντιμετώπιση του προβλήματος, επιλέξαμε στο Virtualmin Server Configuration -> Website Options -> Maximum PHP script run time και τροποποιήσαμε το όριο από 30 στα 60. Έπειτα το αρχείο ανέβηκε επιτυχώς. Παρακαλούμε να το ελέγξετε από πλευράς σας.
