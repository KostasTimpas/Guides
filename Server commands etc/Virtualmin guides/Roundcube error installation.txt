Συνδεόμαστε στο Webmin σαν root και επιλέγουμε από το tab Webmin -> Servers -> MySQL Database Server -> User Permissions. Στον user που αποτυγχάνει το installation με το μήνυμα λάθους access denied for user mysqluser@localhost (συνήθως είναι ο default user του virtual server) θα πρέπει να εισάγετε εκ νέου τον κωδικό του. Τον κωδικό του default user (στο ανωτέρω παράδειγμα mysqluser) μπορείτε να τον ανακτήσετε (στην περίπτωση που δεν είναι κρυπτογραφημένος) επιλέγοντας το tab Virtualmin -> Edit Virtual Server στο hosting που θα εγκατασταθεί το roundcube -> Configurable settings -> Administration password -> και επιλέγετε το εικονίδιο με το κλειδάκι (στην περίπτωση που δεν υπάρχει το εικονίδιο με το κλειδί, σημαίνει ότι ο κωδικός είναι κρυπτογραφημένος και δεν υπάρχει τρόπος ανάκτησης, συνεπώς θα πρέπει να προβείτε στην αλλαγή του επιλέγοντας Set to .. και γράφοντας τον νέο κωδικό).
